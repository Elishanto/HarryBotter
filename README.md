# HarryBotter
[![Build Status](https://travis-ci.org/Elishanto/HarryBotter.svg?branch=master)](https://travis-ci.org/Elishanto/HarryBotter)
> Facebook Messenger bot

## Install dependencies

Install with pip

```sh
$ pip install -r requirements.txt
```

## Running tests

Install nosetests:
```sh
$ pip install nose
```

```sh
$ nosetests tests
```
